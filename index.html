Tired of decompiling minecraft using MCP ? Tired of scrolling a long time to found the java file your looking for between the thousands of minecraft ones ? Tired of managing Minecraft dependencies ? Tired of running compile.bat to test your mod ?

Then MineClipse is for You.

MineClipse is an open source eclipse plugin created to match Minecraft modders's requirements, such as cleaner project, quick glimpse in a view, file overrider, GUI builder or mob maker, with all the avantages of a real and powerful IDE, like auto-completion, quick run and syntax checker : Eclipse IDE.
